# Планировщик задач

## Какие вопросы?
* Как выводить
* Где хранить
* Как будет работать

## Примерный план

### Выводить
* Запустить веб сервер
* Сделать страничку для ввода данных
* Сделать страничку для отображения данных

### Хранить
* Сохранять в файл (в формате json)
* Потом подумать как это перенести в БД
* Подумать какую БД использовать и как поднимать локально это дело

### Как примерно будет работать
* Структура TODO события продумать
    * Уникальный ID
    * Текст что сделать
    * Дата начала
    * Дата конца
    * Статус
    * Приоритет
    * Тег
* Работа с событием
    * Чтение
    * Запись
    * Изменение(это возможно просто запись же)
